# App Android — Fressco (clientes)

Aplicación cliente donde los usuarios finales pueden ver productos próximos
a vencer, agregarlos al carrito, realizar pedidos y ver sus ofertas especiales.

## Stack
- Kotlin
- Retrofit (consumo de la API del backend)

## Pantallas principales
- Iniciar sesión / Registrar cuenta
- Pantalla principal
- Productos
- Carrito de compra
- Formulario de pedido
- Mis pedidos
- Ofertas especiales

## Cómo abrir el proyecto
1. Abrir Android Studio.
2. Abrir esta carpeta (`app-android/`) como proyecto.
3. Configurar la URL base del backend en el archivo de configuración de Retrofit.
