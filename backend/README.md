# Backend — Fressco

API REST que centraliza la lógica de negocio y sirve tanto a la app Android
como a la web de administrador. Se conecta directamente a la base de datos
relacional (ver carpeta `bd/`).

## Stack
- Node.js + Express (o el framework definido por el equipo)
- Conexión a PostgreSQL / MySQL

## Endpoints principales (pendiente de definir)
- `/productos`
- `/usuarios`
- `/ofertas`
- `/pedidos`

## Cómo correr localmente
npm install
npm run dev

## Variables de entorno
Crear un archivo `.env` con la conexión a la base de datos (ver `.env.example`).
